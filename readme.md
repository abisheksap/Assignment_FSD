# Personal Portfolio Website

Welcome to the repository of my custom portfolio website. This project showcases my skills, projects, and professional background in a clean, responsive, and interactive layout.

Built with **HTML5**, **CSS3**, **JavaScript**, and **Bootstrap 5**, this portfolio is fully customizable and ready to be deployed as your personal brand hub.

## Features

- **Fully Responsive** – Looks great on desktops, tablets, and mobiles (Bootstrap grid + custom media queries).
- **Modern & Clean UI** – Carefully chosen color palette, typography, and spacing.
- **Interactive Elements** – Smooth scrolling, hover effects, and dynamic content (JavaScript).
- **About Section** – Your story, skills, and contact information.
- **Projects Gallery** – Showcase your work with images, descriptions, and links.
- **Contact Form** – Functional form with validation (backend integration ready).
- **Fast & Lightweight** – No heavy frameworks, optimized assets.
- **Cross‑Browser Compatible** – Works on Chrome, Firefox, Safari, Edge.
